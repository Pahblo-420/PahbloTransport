## Overview
Scholar Transport is a static website designed to showcase school transportation services, including daily transport, after-school transport, and special trips. It provides clear information about services, safety measures, routes, and contact details to help parents and schools make informed decisions.

![Home Page Screenshot](images/homepage.png)

## Features
- Home page with hero section and introduction
- Services page detailing all transport options
- Safety page describing driver experience, vehicle safety, and compliance
- Routes page showing transport coverage and planning
- Contact section for inquiries and bookings
- Fully responsive design for desktop and mobile devices

## Technologies Used
- HTML5
- CSS3
- Font Awesome (for icons)

## Setup and Usage
1. Clone the repository:
```bash
git clone <repository-url>
```
2. Navigate to the project folder.
3. Open `index.html` in any modern web browser.

No server or installation is required because the website is fully static.

## Customization
You can customize the website by:
- Editing text directly in HTML files
- Replacing images in the `images` folder
- Updating styles in `css/styles.css`
